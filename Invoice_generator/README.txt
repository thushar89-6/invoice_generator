# Invoice Generator Application

The Invoice Generator Application is a Python GUI app built using the tkinter library. It allows users to enter consignee details, destination, weight, and amount, which are then displayed in a table. The app also provides functionality to edit and delete entries, as well as generate an invoice in PDF format.

## Features

- User-friendly interface for entering consignee details, destination, weight, and amount.
- Automatic display of entered values in a table for easy reference.
- Edit functionality to modify any entry in the table.
- Delete functionality to remove the last row from the table.
- Download button to generate an invoice in PDF format based on the entered values.
- Previous Invoice to edit/add/regenerate previous invoices.

## Prerequisites

- Windows operating system

## Usage

- Double-click on the `Invoice_generator.exe or main.py file to start the application.
- Enter the consignee details, destination, weight, and amount in the respective input fields.
- Click the "Add Entry" button to add the values to the table.
- To edit an entry, select the row in the table and click the "Edit" button.
- To delete the last row from the table, click the "Delete Last Row" button.
- To generate the invoice in PDF format, click the "Download" button. The invoice will be saved in the current directory.
- To edit/view/add and regenerate previously generated invoices, click on previous invoice button.
